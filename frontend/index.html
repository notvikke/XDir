Our custom Manifest V3 extension is located inside your application folder. It bridges Chrome's active login session with this desktop application.

Loading extension path...

2. How to Scrape & Sync Metadata

Once loaded in Chrome, visit any game thread on F95Zone or product page on DLsite while logged in.

  • Update Alerts: The extension warns you whenever an online version exceeds your local archive build.

Welcome to XDir Offline Game Manager

1. How Your Game Directory Should Look

XDir automatically scans and organizes your standalone PC games, visual novels, and archives. For best results, organize your configured Primary Game Setups Directory as follows:

  • Individual Folders: Each game should be in its own subfolder containing the executable or game files.
  • Compressed Archives: Standalone .zip, .rar, .7z, or .iso archives are supported directly. XDir can launch or extract them on demand!
  • Category Folders (Optional): You can group games into subfolders like \F95\, \DLsite\, or \Itch\ if you like.

2. Manage & Load the Companion Extension

To unlock automatic metadata scraping (high-res covers, 15+ screenshots, community ratings, and live wishlist syncing from F95zone, DLsite, and Itch.io), install the bundled Chrome extension:

Loading extension path...
  1. Open Google Chrome and navigate to chrome://extensions
  2. Toggle on Developer mode in the top right corner.
  3. Click Load unpacked and select the extension folder shown above.
